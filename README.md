# Server Performance Stats

A simple Bash script that analyzes basic Linux server performance and displays key system statistics in a clean, readable format.

## Features

* Total CPU usage
* Memory usage (used, free, and percentage)
* Disk usage (used, free, and percentage)
* Top 5 processes by CPU usage
* Top 5 processes by memory usage
* OS version
* System uptime
* Load average
* Logged-in users
* Recent failed login attempts (if available)

## Usage

```bash
chmod +x server-stats.sh
./server-stats.sh
```

This project is a lightweight server monitoring utility built using standard Linux commands, making it compatible with most Linux distributions without requiring additional dependencies.
